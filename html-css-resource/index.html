HTML and CSS Quick Reference Guide

This guide explains useful HTML tags and CSS properties for building simple, well-structured web pages.

Common HTML Tags

HTML tags are used to structure the content of a web page. The examples below show common tags and how they are used.

1. Heading Tags: <h1> to <h6>

Heading tags create titles and subtitles. The <h1> tag is the most important heading.

<h1>Main Page Title</h1>
<h2>Section Title</h2>
<h3>Small Heading</h3>

2. Paragraph Tag: <p>

The paragraph tag is used for normal text content.

<p>This is a paragraph of text.</p>

3. Link Tag: <a>

The anchor tag creates a link to another page or website.

<a href="https://www.example.com">Visit Example</a>

4. Image Tag: <img>

The image tag displays an image on a web page.

<img src="image.jpg" alt="Description of the image">

5. Unordered List: <ul> and <li>

An unordered list displays bullet points.

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

6. Ordered List: <ol> and <li>

An ordered list displays numbered steps.

<ol>
    <li>Open the file</li>
    <li>Edit the code</li>
    <li>Save the page</li>
</ol>

7. Division Tag: <div>

The division tag is a container used to group elements together.

<div class="card">
    <h2>Card Title</h2>
    <p>Card content goes here.</p>
</div>

8. Semantic Section Tag: <section>

The section tag groups related content in a meaningful way.

<section>
    <h2>About This Topic</h2>
    <p>Information about the topic.</p>
</section>

9. Article Tag: <article>

The article tag is used for independent content, such as a blog post or guide item.

<article>
    <h2>News Title</h2>
    <p>News content goes here.</p>
</article>

10. Footer Tag: <footer>

The footer tag usually contains page information, copyright text, or extra links.

<footer>
    <p>Created by a web development student.</p>
</footer>

Useful CSS Properties

CSS properties control the appearance of HTML elements. These examples show common properties used in web design.

color

Changes the colour of text.

background-color

Changes the background colour of an element.

font-family

Sets the typeface used for text.

font-size

Controls the size of text.

margin

Adds space outside an element.

padding

Adds space inside an element.

border

Adds a line around an element.

border-radius

Rounds the corners of an element.

display

Controls how elements are arranged on the page.

box-shadow

Adds a shadow effect behind an element.

body {
    font-family: Arial, sans-serif;
    background-color: #f4f6f8;
    color: #222222;
}

.card {
    margin: 20px;
    padding: 15px;
    border: 1px solid #cccccc;
    border-radius: 10px;
}

Simple HTML Page Structure

A good HTML page normally includes a clear structure with <header>, <main>, and <footer>. 

<body>
    <header>
        <h1>Website Title</h1>
    </header>

    <main>
        <section>
            <h2>Main Section</h2>
            <p>Page content goes here.</p>
        </section>
    </main>

    <footer>
        <p>Footer information</p>
    </footer>
</body>
